CODE-CREW Installation Guide

Required Software

| Software | Version |
|----------|----------|
| Python | 3.10+ |
| MySQL | 8.0+ |
| VS Code | Latest |
| Git | Latest |

---

Clone Project

```bash
git clone https://github.com/yourusername/CODE-CREW.git
cd CODE-CREW
```

---

Create Virtual Environment

```bash
python -m venv venv
```

---

Activate Virtual Environment

Windows
```bash
venv\Scripts\activate
```

Install Modules

```bash
pip install -r requirements.txt
```
